# Day 1 - Bash BASICS

## What is Bash??
Bash is the most commonly used shell in Ubuntu. it interprets commands and can 
can execute scripts(.sh) 
#Shell: 
It means a command line interpreter, where we can run commands, Can 
execute commands , it parses what we write and talks to KERNEL.

##Commands Used:
- echo
- $USER
- $HOME
- $PATH
- chmod +x
- ./script.sh   (here why ./) because script.sh is not found in $PATH so
must execute from current directory 
 
##Script Created:
script is created with nano hello.sh (text file) and At the start we wrote 
#!/bin/bash : means shebang , tells linux to use this to execute the file
then it is made executable with chmod +x command
